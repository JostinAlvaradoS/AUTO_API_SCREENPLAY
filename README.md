# AUTO_API_SCREENPLAY

Automatización de servicios REST utilizando **Screenplay Rest**.

## 🎯 Objetivo
Validar el ciclo completo (CRUD) de la API de Reservas/Inventario.

## 🛠️ Verbos a validar
- **POST**: Crear reserva.
- **GET**: Consultar estado.
- **PUT**: Actualizar configuración.
- **DELETE**: Cancelar/Liberar reserva.

## 🚀 Ejecución
```bash
./gradlew clean test aggregate
```
